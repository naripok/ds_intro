Redes Neurais Artificiais, Tensorflow e Keras¶

No contexto de modelagem e aprendizado de mquina, as redes neurais artificiais profundas vem tomando seu lugar no estado de arte como tecnicas de modelagem extremamente poderosas. Suas propriedaes intrnsecas permitem a estas estruturas aprenderem representaes hierarquicas de alto nvel baseadas nos dados de treinamento. Esta capacidade torna as redes neurais profundas timas estruturas de compresso de dados, com possibilidade de otimizao utilizando propagao reversa dos gradientes (modelo diferenciavel), o que torna o seu treinamento eficiente e de rpida convergencia.

O Tensorflow uma biblioteca da Google com bindings em Python, capaz de gerar e computar grafos matemticos com altssima eficincia. Esta biblioteca poderosssima esta em constante evoluo e vem sendo empregada em todos os setores da indstria.

O Keras nada mais do que um modulo de alto nvel por cima do Tensorflow, muito bom para prototipagem rpida e implementao de redes neurais artificiais.

Nesta sesso faremos uso do Keras para implementar redes neurais artificiais e utiliza-las para fazer predies sobre problemas de Supervised Learning.

Vamos utilizar para demonstrao o clssico dataset MNIST. Este conjunto de dados consiste em imagens de digitos manuscritos de 0 a 9, com os respectivos rtulos associados. Estes ultimos sero utilizados como sinal de superviso. Nosso objetivo aprender o mapeamento entre os pixels da imagem e a respectiva classe (0 a 9) ao qual aquele digito pertence.

Multilayer Perceptron¶

In [6]:
np.random.seed(42)                                      # fixa a random seed para obter repetibilidade dos resultados

def mlp(n_in, n_hidden, n_out):                         # Funao de inicializaao das variaveis da rede neural

    U = np.random.normal(size=(n_hidden, n_in))         # Pesos da primeira layer (Distribuiao normal com desvio 0.1 e shape (n_in, n_hidden)
    V = np.random.normal(size=(n_hidden, n_hidden))     # O mesmo para a segunda layer
    W = np.random.normal(size=(n_out, n_hidden))        # pesos layer de saida

    bu = np.zeros(n_hidden)                             # Inicializa bias da primeira layer
    bv = np.zeros(n_hidden)                             # mesmo segunda layer
    bw = np.zeros(n_out)                                # Bias de sada

    params = [U,V,W,bu,bv,bw]                           # transforma os parametros em lista de parametros

    return params                                       # retorna parametros
  • A segunda funo a funo de ativao dos neuronios a ser utilizada. Neste caso, usaremos a funo sigmoid.
In [7]:
class Sigmoid:
    @classmethod
    def forward(cls, X):
        return 1.0 / (1.0 + np.exp(-X))
    
    @classmethod
    def backward(cls, X):
        output = cls.forward(X)
        return (1.0 - output) * output

def one_hot(targets, nb_classes):
    res = np.eye(nb_classes)[np.array(targets).reshape(-1)]
    return res.reshape(list(targets.shape)+[nb_classes])
  • A terceira funo a funo de propagao direta da nossa rede neural, e ser usada para fazer inferncia.
In [8]:
def predict(x, U, V, W, bu, bv, bw):    # Realiza predioes com a rede neural
    A = np.dot(U, x) + bu               # Calcula a sada da layer 1 sem ativaao
    B = np.dot(V, Sigmoid.forward(A)) + bv      # Calcula a saida da layer de dois sem ativaao
    C = np.dot(W, Sigmoid.forward(B)) + bw      # saida layer, saida sem ativaao
    Z = Sigmoid.forward(C)                      # ativaao layer saida
    index = np.argmax(Z)                # adquire o index no array de 1 a 10 que corresponde ao valor da prediao
    out = np.zeros((1, 10))             # cria array de dez valores 0
    out[0, index] = 1                   # atribui valor 1 ao valor no array cujo index corresponde a sada da layer
    return Z, out.astype(int)           # retorna probabilidade de cada valor, prediao em one-hot

Como medida de custo, vamos utilizar a entropia relativa entre os as distribuies de probabilidade das nossas previses e dos dados de referncia.

def cross_entropy(y_hat, y):
    return -np.mean(y_hat * np.log(y) + (1 - y_hat) * np.log(1 - y))
  • A quarta e ltima funo a funo de propagao reversa da rede neural. Esta funo trata de calcular os gradientes do erro em relao aos parmetros do modelo. Utilizar estes gradientes para atualizar os pesos do modelo em questo chamado Gradient Descent. Quando ele feito em amostrar menores do dataset, ele se torna o Stochastic Gradient Descent.
In [10]:
def batch_generator(x, y, batch_size=64, n_out=10):
    idx = 0
    while True:
        yield x[idx:idx+batch_size].reshape((-1, 784)), one_hot(y[idx:idx+batch_size], n_out)
        idx += batch_size
        if idx >= x.shape[0]:
            idx=0

def train(x, y, U, V, W, bu, bv, bw): #funao treinamento
    """
    PARAMETROS: x = input data
                V =  Pesos da primeira layer
                W =  Pesos da segunda layer
                bv = Bias da primeiralayer
                bw = Bias da segunda layer

    """
    # forward propagation
    A = np.dot(U, x) + bu   # realiza a multiplicaao escalar da entrada pelos pesos da primeira layer
                            # e adidiona o valor do bias

    B = np.dot(V, Sigmoid.forward(A)) + bv   # realiza a multiplicaao matricial da saida da ativaao sigmoid da primeira layer
                                     # pelos pesos da segunda layer e adiciona os bias para esta layer
        
    C = np.dot(W, Sigmoid.forward(B)) + bw   # mesmo com a layer de saida
    
    Y = Sigmoid.forward(C)                   # ativaao da layer de saida
    

    # backward propagation (https://en.wikipedia.org/wiki/Backpropagation)
    Ew = Y - y                                # Diferena do valor da sada predita pela sada esperada (gradiente)
    Ev = Sigmoid.backward(B) * np.dot(Ew, W)  # Calcula o erro para a layer DOIS (Gradiente do erro * pesos da segunda layer) * sigmoid(hidden))
    Eu = Sigmoid.backward(A) * np.dot(Ev, V)  # mesmo para layer um
    
    # Deltas utilizados para atualizar os pesos da rede
    dW = np.outer(Ew, Sigmoid.backward(B))    # DeltaW, Calcula a distributiva de cada termo do array e retorna lista de arrays distribuidos
    dV = np.outer(Ev, Sigmoid.backward(A))    # DeltaV  , Hadamard product
    dU = np.outer(Eu, x)                      # mesmo
    loss = cross_entropy(y, Y) #funao loss = cross-entropy

    return  Y, loss, (dU, dV, dW, Eu, Ev, Ew)  #retorna o valor da funao custo e o gradiente do erro em funao dos pesos e bias

O cdigo a seguir implementa a rotina de treinamento do modelo no dataset em amostras, ou batches. Aqui, chamamos a funo train em cada amostra e acumulamos os gradientes durante um batch. Atualizamos ento os pesos do nosso modelo com a subtrao dos gradientes mais um valor de momento para acelerar o treinamento. Este algoritmo uma das formas mais simples de Stochastic Gradient Descent e variaes mais poderosas esto disponveis na literatura e implementadas e prontas para uso em modulos de amprendizado de mquina.

import time

# Parametros de treinamento
epochs = 100
batch_size = 128
learning_rate = 0.001
momentum = 0.99
n_hidden = 256  #numero de neuronios nas hidden layers

# Treinamento
print("Start Training Session")

train_err = []
train_accu = []
eval_err = []
eval_accu = []

mnist_train = batch_generator(x_train, y_train, batch_size)
mnist_eval = batch_generator(x_val, y_val, batch_size)

# Adquire dados
image_sample, label_sample = next(mnist_train)

# VARIAVEIS DA DNN
n_in = image_sample.shape[1]      #numero de features de entrada, neste caso o numero de pixels da imagem
n_out = label_sample.shape[1]     #numero de sadas do grapho, neste caso 10 classes diferentes
n_samples = x_train.shape[0] - 1  #numero de exemplos no dataset

params = mlp(n_in, n_hidden, n_out)    #inicializa variaveis da rede neural

for epoch in range(epochs):
    
    X, Y = next(mnist_train)
    X_eval, Y_eval = next(mnist_eval)
    
    err = []                                # placeholder da lista de errors
    upd = [0]*len(params)                   # placeholder dos parametros de atualizaao dos pesos
    corr_pred = []                          # Bookkeeping
    
    for i in range(batch_size):                        # realiza treinamento
        pred, loss, grad = train(X[i], Y[i], *params)  # chama a funao train, o *params 'explode' a lista de parmetros

        for j in range(len(params)):                   # faz o update dos parametros da rede
            params[j] -= upd[j]

        for j in range(len(params)):                   # calcula o valor a ser atualizado nos parametros para o proximo ciclo
            upd[j] = learning_rate * grad[j] + momentum * upd[j]    # mi*gradiente+momento*update, gradiente com momento
        # Bookkeeping
        err.append(loss)                                            # adquire o custo em lista de errors
        corr_pred.append(np.argmax(Y[i]) == np.argmax(pred))        # adiciona corretude da prediao em lista
    
   
     
    if (epoch + 1) % 10 == 0 or epoch == 0:
        accuracy = np.mean(corr_pred) * 100     # calcula a precisao
        print("Train Epoch {0} -> Loss: {1}, Accuracy: {2} %"
              .format(epoch, np.mean(err), np.mean(corr_pred) * 100))
    
        train_err.append(err)
        train_accu.append(corr_pred)
    
        err = []
        corr_pred = []

        for i in range(batch_size):
            Z, pred = predict(X_eval[i], *params)   # chama a funao predict para fazer predioes com entradas
            t_ = Y_eval[i]                          # matriz de labels real para clculo perdas
            eval_loss = cross_entropy(t_, Z)          

            # Bookkeeping
            err.append(eval_loss)
            corr_pred.append(np.argmax(Y_eval[i]) == np.argmax(pred))
        eval_err.append(err)
        eval_accu.append(corr_pred)    

        print("Eval  Epoch {0} -> Loss: {1}, Accuracy: {2} %"
                  .format(epoch, np.mean(err), np.mean(corr_pred) * 100))
