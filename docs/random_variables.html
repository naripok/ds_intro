Variáveis Aleatórias¶

Uma varivel aleatria aquela que assume valores de acordo com dada chance. Ao discutir variveis aleatrias, normalmente as descrevemos em termos de distribuies de probabilidade. Ou seja, a probabilidade de cada valor possa ser amostrado da varivel aleatria. O exemplo clssico disso um dado, que pode produzir os valores 1-6 com probabilidade uniforme.

Normalmente, separamos variveis aleatrias em duas classes diferentes:

  • Variveis aleatrias discretas
  • Variveis aleatrias contnuas

Cada varivel aleatria segue uma distribuio de probabilidade, uma funo que a descreve. A distribuio de probabilidade atribui probabilidades a todos os valores possveis de uma varivel aleatria. Para uma determinada varivel aleatria $X$, expressamos a probabilidade de que $X$ seja igual a um valor $x$ como $P (X = x)$. Para variveis aleatrias discretas, podemos expressar $p (x) = P (X = x)$. Isso tambm conhecido como a funo massa de probabilidade (PMF). Para variveis aleatrias contnuas, no podemos usar um PMF, como abordaremos mais tarde, e devemos utilizar a funo densidade de probabilidade (PDF) em seu lugar. As distribuies de probabilidade formam a base para modelos estatsticos descritivos. A compreenso delas tambm necessria para realizar simulaes de Monte Carlo.

Para cada funo de distribuio de probabilidade, tambm temos uma funo distribuio acumulada (CDF). Isso definido como $P (X x)$, a probabilidade de que a varivel aleatria seja menor ou igual a um valor especfico. A abreviatura para o CDF $F (x) = P (Xx)$. Para encontrar $F (x)$ no caso discreto, resumimos os valores do PMF para todos os resultados inferiores ou iguais a $x$. No caso contnuo, usamos o clculo para integrar o PDF em todos os valores at $x$.

Variáveis Aleatórias Discretas¶

Uma varivel aleatria discreta uma com um nmero finito de resultados. Cada um desses resultados tem uma probabilidade nica associada. Considere uma moeda justa ou um dado, algumas das variveis aleatrias uniformemente distribudas mais bsicas. Para a moeda, existem dois resultados possveis, seja cara ou coroa, cada uma com uma probabilidade $ 1/2 $ de ocorrncia. Variveis aleatrias discretas nem sempre tm pesos iguais para todos os resultados. A unidade bsica de uma varivel aleatria discreta a funo massa de probabilidade (PMF), outro nome para a funo de probabilidade $ p (x) $. O PMF, ou funo de probabilidade, d uma probabilidade, uma massa, a cada ponto no domnio da distribuio de probabilidade. Uma funo de probabilidade tem duas propriedades principais:

  1. $ 0p(x)1 $ porque todas as probabilidades esto no intervalo $ [0, 1] $
  2. A soma de todas as probabilidades de $ p (x) $ em todos os valores de $X$ igual a $ 1 $. Os pesos totais para todos os valores da varivel aleatria devem somar $ 1 $.

Aqui vamos considerar alguns exemplos das distribuies de probabilidade discreta mais revalentes.

class DiscreteRandomVariable:
    def __init__(self, a=0, b=1):
        self.variableType = ""
        self.low = a
        self.high = b
        return
    def draw(self, numberOfSamples):
        samples = np.random.randint(self.low, self.high + 1, numberOfSamples)
        return samples

Distribuição uniforme¶

O tipo mais bsico de distribuio de probabilidade a distribuio uniforme. Com uma distribuio uniforme discreta, o mesmo peso atribudo a todos os resultados. Tome o exemplo de rodar um dado. Ele tem seis faces, numeradas $ 1 $ at $ 6 $, cada uma com probabilidade de ocorrer com chance de $ 1/6 $. Com isso, sabemos que o PMF deve ser $ p (x) = 1/6 $ para todos os valores de nossa varivel aleatria uniforme $ X $.

Usando esta tabela, podemos ver facilmente que a funo de probabilidade satisfaz as condies necessrias. Cada valor da funo de probabilidade est no intervalo $ [0,1] $, satisfazendo a primeira condio. A segunda condio satisfeita porque todos os valores de $ p (x) $ soma para $ 1 $, conforme evidenciado na funo de distribuio cumulativa. Isto mostra duas propriedades da funo de distribuio cumulativa:

  1. O CDF est entre $ 0 $ e $ 1 $ para todos os $ x $. Isso condizente com o valor da funo de distribuio de probabilidade.
  2. O CDF no decrescente em $ x $. Isso significa que, medida que $ x $ aumenta, o CDF aumenta ou permanece constante.

Ao tentar amostrar outras distribuies de probabilidade, podemos usar composies da distribuio uniforme com certas funes para obter as amostras apropriadas. No entanto, esse mtodo pode ser tremendamente ineficiente. Como tal, usaremos as funes NumPy incorporadas para cada distribuio.

Variables aleatórias contínuas¶

As variveis aleatrias contnuas diferem das variveis aleatrias discretas, na medida em que as contnuas podem levar infinitos resultados diferentes. Eles no podem ser contados ou descritos como uma lista. Como tal, significa muito pouco quando atribumos probabilidades individuais aos resultados. Porque h infinitos resultados possveis, a probabilidade de atingir qualquer resultado individual tem uma probabilidade de 0.

Podemos resolver este problema tomando probabilidades em intervalos de valores. Isso feito usando o clculo, no entanto, para usar nossas tcnicas de amostragem aqui, no precisamos usar nenhum. Para uma varivel aleatria contnua $ P (X = 0) $ no temos resposta clara. Ao invs disso, procuraramos algo mais como $ P (-1 <X <1) $. Para variveis aleatrias contnuas, em vez de usar um PMF, definimos uma funo de densidade de probabilidade (PDF), $ f_X (x) $, de modo que podemos dizer: $$ P (a <X <b) = \int_a ^ b f_X (x) dx $$

Semelhante ao nosso requisito para distribuies discretas de que todas as probabilidades somem $ 1 $, aqui exigimos que:

  1. $f_X(x) \geq 0$ para todo valor de $X$
  2. $P(-\infty < X < \infty) = \int_{-\infty}^{\infty} f_X(x) dx = 1$

Vale ressaltar que, porque a probabilidade em um ponto individual com uma distribuio contnua de $ 0 $, as probabilidades nos pontos finais de um intervalo so $ 0 $. Portanto, $ P (a \leq X \leq b) = P (a <X \leq b) = P (a \leq X <B) = P (a <X <b) $. Se integrarmos o PDF em todas as possibilidades, no valor total possvel, o valor deve ser $ 1 $.

class ContinuousRandomVariable:
    def __init__(self, a = 0, b = 1):
        self.variableType = ""
        self.low = a
        self.high = b
        return
    def draw(self, numberOfSamples):
        samples = np.random.uniform(self.low, self.high, numberOfSamples)
        return samples

Distribuição uniforme¶

A distribuio uniforme tambm pode ser definida dentro do quadro de uma varivel aleatria contnua. Ns levamos $ a $ e $ b $ para ser constante, onde $ b $ o valor mais alto possvel e $ a $ o menor valor possvel que o resultado pode obter. Ento o PDF de uma varivel aleatria uniforme :

$$f(x) = \begin{cases}\frac{1}{b - a} & \text{for $a < x < b$} \ ,0 & \text{do contrrio}\end{cases}$$

Uma vez que esta funo definida em um intervalo contnuo, o PDF cobre todos os valores entre $ a $ e $ b $. Aqui temos um grfico do PDF (sinta-se livre para variar os valores de $ a $ e $ b $):

$$ F(x) = \begin{cases} 0 & \text{for $x \leq a$} \ \frac{x - a}{b - a} ,& \text{para $a < x < b$} \ 1 ,& \text{para $x \geq b$}\end{cases}$$

Distribuição normal¶

A distribuio normal uma distribuio muito comum e importante em estatstica. Muitos testes e mtodos importantes em estatstica so baseados na suposio de normalidade. Uma grande parte disso devido aos resultados do Teorema do Limite Central (CLT), que afirma que amostras suficientemente grandes de ensaios independentes so normalmente distribudos.

In [8]:
class NormalRandomVariable(ContinuousRandomVariable):
    def __init__(self, mean = 0, variance = 1):
        ContinuousRandomVariable.__init__(self)
        self.variableType = "Normal"
        self.mean = mean
        self.standardDeviation = np.sqrt(variance)
        return
    def draw(self, numberOfSamples):
        samples = np.random.normal(self.mean, self.standardDeviation, numberOfSamples)
        return samples

Ao descrever uma varivel aleatria normal, s precisamos saber sua mdia ($ \mu $) e varincia ($ \sigma ^ 2 $, onde $ \sigma $ o desvio padro). Denotamos uma varivel aleatria $ X $ como uma varivel aleatria normal, dizendo $ X $ ~ $ N (\mu, \sigma ^ 2) $. Uma das principais caractersticas de uma varivel aleatria normal que uma combinao linear de duas ou mais variveis aleatrias normais outra varivel aleatria normal. At este ponto, consideramos apenas distribuies de probabilidade variveis nicas ou univariadas. Quando queremos descrever variveis aleatrias ao mesmo tempo, podemos olhar para uma distribuio multivariada. Uma distribuio normal multivariada descrita inteiramente pelas mdias de cada varivel, suas varincias e as correlaes distintas entre cada par de variveis.

O PDF de uma varivel aleatria normal :

$$ f(x) = \frac{1}{\sigma\sqrt{2\pi}}e^{-\frac{(x - \mu)^2}{2\sigma^2}} $$

E definido por $ - \infty <x <\infty $. Quando temos $ \mu = 0 $ e $ \sigma = 1 $, chamamos isso de distribuio normal padro.

Ao alterar o desvio padro e mdio da distribuio normal, podemos alterar a profundidade e a largura da curva. Com um desvio padro maior, os valores da distribuio so menos concentrados em torno da mdia. Dada a mdia e varincia de uma distribuio normal, podemos fazer as seguintes afirmaes:

  • Cerca de $68\%$ de todas as amostras encontram-se um desvio padro ao redor da mdia ($\mu \pm \sigma$)
  • Cerca de $95\%$ de todas as amostras encontram-se dentro de duas vezes o desvio padro ao redor da mdia ($\mu \pm 2\sigma$)
  • Cerca de $99\%$ de todas as amostras encontram-se dentro de trs vezes o desvio padro ao redor da mdia ($\mu \pm 3\sigma$)

Esses valores so importantes para a compreenso dos intervalos de confiana em relao distribuio normal.

Usando o teorema do limite central, podemos padronizar variveis aleatrias diferentes para que elas se tornem variveis aleatrias normais. Uma ferramenta muito comum nas estatsticas uma tabela de probabilidade padro normal, usada para pesquisar os valores da CDF padro normal para valores dados de $ x $. Ao transformar variveis aleatrias em um padro normal, podemos simplesmente verificar essas tabelas para valores de probabilidade. Ns padronizamos uma varivel aleatria $ X $ subtraindo a mdia e a diviso pela varincia, resultando na varivel aleatria normal padro $ Z $.

$$ Z = \frac{X - \mu}{\sigma} $$

Vejamos o caso em que $ X $ ~ $ B (n, p) $ uma varivel aleatria binomial. No caso de uma varivel aleatria binomial, a mdia $ \ mu = np $ e a varincia $ \ sigma ^ 2 = np (1 - p) $.

class BinomialRandomVariable(DiscreteRandomVariable):
    def __init__(self, numberOfTrials = 10, probabilityOfSuccess = 0.5):
        self.variableType = "Binomial"
        self.numberOfTrials = numberOfTrials
        self.probabilityOfSuccess = probabilityOfSuccess
        return
    def draw(self, numberOfSamples):
        samples = np.random.binomial(self.numberOfTrials, self.probabilityOfSuccess, numberOfSamples)
        return samples

Ajustando uma distribuição¶

Agora, tentaremos ajustar uma distribuio de probabilidade. Tomaremos os retornos de Apple e tentaremos adequ-los a uma distribuio normal. A primeira coisa a verificar se os retornos realmente exibem propriedades de uma distribuio normal. Para este propsito, usaremos o teste Jarque-Bera, que indica no-normalidade se o valor p estiver abaixo de um ponto de corte.

Uma maior segmentao leva a um pico mais elevado. No entanto, extremamente difcil encontrar dados do mundo real que se encaixem bem nas suposies de normalidade.

Geralmente, ao tentar ajustar uma distribuio de probabilidade aos valores do mundo real, devemos ter uma distribuio particular (ou distribuies) em mente. H uma variedade de testes para diferentes distribuies que podemos aplicar para ver o que poderia ser o melhor. Alm disso, medida que mais informaes estiverem disponveis, ser necessrio atualizar a mdia da amostra e o desvio padro ou talvez at encontrar uma distribuio diferente para refletir mais precisamente as novas informaes. A forma da distribuio mudar em conformidade.
