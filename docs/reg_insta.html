Instabilidade em Modelos Regressores¶

A anlise de regresso nos permite estimar coeficientes em uma funo que relaciona aproximadamente vrios conjuntos de dados. Ns levantamos a hiptese de uma forma especfica para esta funo e, em seguida, encontramos coeficientes que se encaixam bem nos dados, trabalhando sob o pressuposto de que os desvios do modelo podem ser considerados rudos.

Ao construir esse modelo, aceitamos que no podemos prever perfeitamente a varivel dependente. Aqui, gostaramos de avaliar a preciso do modelo, no pelo quo bem ele explica a varivel dependente, mas por quo estvel (ou seja, o quo estvel so os coeficientes de regresso) em relao nossa amostra dados. Afinal, se um modelo tiver realmente um bom ajuste, deve ser semelhante, digamos, para duas amostras aleatrias de nosso conjunto de dados, modeladas individualmente. Caso contrrio, no podemos assumir que o modelo no simplesmente um artefato da amostra particular de dados que ns escolhemos (overfiting), ou que far boas previses para novos pontos de dados.

Usaremos regresses lineares aqui para fins de ilustrao, mas as mesmas consideraes se aplicam a todos os modelos de regresso. Abaixo, definimos uma funo de wrapper para a regresso linear de statsmodels para que possamos us-lo mais tarde.

def linreg(X,Y):
    # Running the linear regression
    x = sm.add_constant(X) # Add a row of 1's so that our model has a constant term
    model = regression.linear_model.OLS(Y, x).fit()
    return model.params[0], model.params[1] # Return the coefficients of the linear model

Viés¶

A amostra particular que escolhemos para os dados afeta o modelo gerado e o rudo distribudo de forma desigual pode levar a um modelo impreciso. Abaixo, amostramos de uma distribuio normal, mas como tomamos poucas amostras, obtemos um significativo desvio para baixo. Se tomarmos mais amostras, ambos os coeficientes de regresso se moveriam em direo a zero.

Regime dinâmico¶

Uma mudana de regime (ou ruptura estrutural) quando algo muda no processo gerador dos dados, fazendo com que futuras amostras sigam uma distribuio diferente. Abaixo, exemplificamos este tipo de evendo utilizando um processo de Ornstein-Uhlenbeck com dois conjuntos de parmetros diferentes, e a diviso dos dados resulta em um ajuste muito melhor (em vermelho) do que uma regresso em todo o conjunto de dados (amarelo). Nesse caso, nosso modelo de regresso no ser preditivo de futuros pontos de dados, j que o sistema subjacente no mais o mesmo que na amostra. Na verdade, a anlise de regresso pressupe que os erros no esto correlacionados e tm variao constante, o que muitas vezes no o caso se houver uma mudana de regime.

  class OrnsteinUhlenbeckProcess(object):
    def __init__(self, theta, mu=0., sigma=1., dt=1e-2, x0=None, size=1, sigma_min=None, n_steps_annealing=100):
        self.mu = mu
        self.sigma = sigma
        
        if sigma_min is not None:
            self.m = -float(sigma - sigma_min) / float(n_steps_annealing)
            self.c = sigma
            self.sigma_min = sigma_min
        else:
            self.m = 0.
            self.c = sigma
            self.sigma_min = sigma
        
        self.n_steps = 0
        self.theta = theta
        self.dt = dt
        self.x0 = x0
        self.size = size
        self.reset_states()

    @property
    def current_sigma(self):
        sigma = max(self.sigma_min, self.m * float(self.n_steps) + self.c)
        return sigma

    def sample(self):
        x = self.x_prev + self.theta * (self.mu - self.x_prev) * self.dt + self.current_sigma * np.sqrt(self.dt) * np.random.normal(size=self.size)
        self.x_prev = x
        self.n_steps += 1
        return x

    def reset_states(self):
        self.x_prev = self.x0 if self.x0 is not None else np.zeros(self.size)

Multicollinearidade¶

Acima, estvamos considerando apenas regresses de uma varivel dependente e uma independente. No entanto, tambm podemos ter mltiplas variveis independentes. Isso leva instabilidade se as variveis independentes estiverem altamente correlacionadas.

Imagine que estamos usando duas variveis independentes, $ X_1 $ e $ X_2 $, que so altamente correlacionadas. Ento, os coeficientes podem mudar drasticamente se adicionarmos uma nova observao que ligeiramente melhor explicada por um dos dois do que pelo outro. No caso extremo, se $ X_1 = X_2 $, a escolha dos coeficientes depender do algoritmo de regresso linear particular.
