Machine Learning e o Sklearn¶

Vamos agora iniciar os trabalhos no campo do aprendizado de maquina e modelagem. Uma habilidade importantissima na rea da cincia de dados a capacidade de compresso e aferimento de informaes. Dado uma esrutura de dados arbitrria, gostariamos de poder inferir informaes desta estrutura de forma eficiente e correta. Para isso, utilizamos de metodos de compresso de dados, como modelos matemticos, para comprimir esta informao de uma forma que torne sua aferio mais eficinte do que se tivessemos que procurar nos dados pelos K vizinhos mais proximos, por exemplo. Muitos algoritmos diferentes de compresso e aferio estao disponveis, e cada tarefa tem algum algoritmo timo que lhe serve. Neste contexto, o "No Free Lunch theorem" diz que, a priori, nenhum algoritmo de machine learning ser melhor do que os outros, e apenas o teste pode revelar o algoritmo timo para dada tarefa.

Dentre os problemas envolvendi machine learning, fazemos trs subdivises:

  • Supervised Learning, ou aprendizado supervisionado
  • Unsupervised Learning, ou aprendizado no supervisionado
  • Reinforcement Learning, ou aprendizado por reforo

Problemas de apendizado de funes de mapeamento entre variveis de entrada e variveis alvo com sinal de superviso se enquandram na classe de Supervised Learning, e deste tipo de problema que trataremos a seguir. Como curiosidade introdutria, algoritmos de Unsupervised Learning aprendem classificaes (ou redues) de pontos de dados a partir de suas propriedades inerentes e sem sinal de superviso (varivel alvo). J os algoritmos destinados a resolver problemas de Reinforcement Learning aprendem representaes de valor de estados do ambiente e politicas de agentes baseados em sinais de reforo obtidos do ambiente no qual estes agentes esto inseridos.

Dentre os problemas presentes no campo do apredizado supervisionado fazemos duas subdivises, so elas problemas de regresso e de classificao. Modelos de regresso tem como sada variveis geralmente continuas, e tentam prever o valor destas variveis dadas variveis de entradas independentes. Modelos de classificao, dadas determinadas entradas, tem como sada a probabilidade de dada amostra pertencer um classe Y dentre N possveis classes.

Iniciaremos declarando as funes auxiliares ao nosso processo.

  • A primeira funo define a estrutura do nosso modelo e inicializa suas variveis. Uma grande variedade de inicializaes pseudo-aleatrias dos pesos est disponvel na literatura e o modo como se inicializa os pesos de uma rede neural profunda afeta como ela aprende. Multilayer Perceptron¶

In [11]:
np.random.seed(42)                                                  # fixa a random seed para obter repetibilidade dos resultados

def mlp(n_in, n_hidden, n_out):                                     # Funao de inicializaao das variaveis da rede neural

    U = np.random.normal(size=(n_in, n_hidden))          # Pesos da primeira layer (Distribuiao normal com desvio 0.1 e shape (n_in, n_hidden)
    V = np.random.normal(size=(n_hidden, int(n_hidden))) # O mesmo para a segunda layer
    W = np.random.normal(size=(int(n_hidden), n_out))    # pesos layer de saida

    bu = np.zeros(n_hidden)                                         # Inicializa bias da primeira layer
    bv = np.zeros(int(n_hidden))                                    # mesmo segunda layer
    bw = np.zeros(n_out)                                            # Bias de sada

    params = [U,V,W,bu,bv,bw]                                       # transforma os parametros em lista de parametros

    return params                                                   # retorna parametros
  • A segunda funo a funo de ativao dos neuronios a ser utilizada. Neste caso, usaremos a funo sigmoid.
In [12]:
def sigmoid(x):                                                     # Funao de ativao dos neurons
    out = []
    for i in range(x.shape[0]):                                     # faz a sigmoid de cada valor no array
        sig_i = 1.0 / (1.0 + np.exp(-x[i]))
        out.append(sig_i)                                           # adiciona o valor de saida da sig a uma lista
                                                                    # de valores para retornar
    return np.array(out)
  • A terceira funo a funo de propagao direta da nossa rede neural, e ser usada para fazer inferncia.
In [13]:
def predict(x, U, V, W, bu, bv, bw):    # Realiza predioes com a rede neural
    A = np.dot(x, U) + bu               # Calcula a sada da layer 1 sem ativaao
    B = np.dot(sigmoid(A), V) + bv      # Calcula a saida da layer de dois sem ativaao
    C = np.dot(sigmoid(B), W) + bw      # saida layer, saida sem ativaao
    Z = sigmoid(C)                      # ativaao layer saida
    index = np.argmax(Z)                # adquire o index no array de 1 a 10 que corresponde ao valor da prediao
    out = np.zeros((1, 10))             # cria array de dez valores 0
    out[0, index] = 1                   # atribui valor 1 ao valor no array cujo index corresponde a sada da layer
    return Z, out.astype(int)           # retorna probabilidade de cada valor, prediao em one-hot
  • A quarta e ltima funo a funo de propagao reversa da rede neural. Esta funo trata de calcular os gradientes do erro em relao aos parmetros do modelo. Utilizar estes gradientes para atualizar os pesos do modelo em questo chamado Gradient Descent. Quando ele feito em amostrar menores do dataset, ele se torna o Stochastic Gradient Descent.
In [14]:
def train(x, y, U, V, W, bu, bv, bw): #funao treinamento

    """
    PARAMETROS: x = input data
            V =  Pesos da primeira layer
            W =  Pesos da segunda layer
            bv = Bias da primeiralayer
            bw = Bias da segunda layer

    """

    t = np.transpose(y) # transpoe a matriz de entrada para realizar as multiplicaoes matriciais

    # forward propagation
    A = np.dot(x, U) + bu   # realiza a multiplicaao escalar da entrada pelos pesos da primeira layer
                            # e adidiona o valor do bias

    B = np.dot(sigmoid(A), V) + bv   # realiza a multiplicaao matricial da saida da ativaao sigmoid da primeira layer
                                     # pelos pesos da segunda layer e adiciona os bias para esta layer

    C = np.dot(sigmoid(B), W) + bw   # mesmo com a layer de saida

    Y = sigmoid(C)                   # ativaao da layer de saida


    # backward propagation
    Ew = Y - t                       # Diferena do valor da sada predita pela sada esperada (gradiente)
    Ev = sigmoid(B) * np.dot(W, Ew)  # Calcula o erro para a layer DOIS (Gradiente do erro * pesos da segunda layer) * sigmoid(hidden))
    Eu = sigmoid(A) * np.dot(V, Ev)  # mesmo para layer um
    
    # Deltas utilizados para atualizar os pesos da rede
    dW = np.outer(sigmoid(B), Ew)    # DeltaW, Calcula a distributiva de cada termo do array e retorna lista de arrays distribuidos
    dV = np.outer(sigmoid(A), Ev)    # DeltaV  , Hadamard product
    dU = np.outer(x, Eu)             # mesmo
    loss = -np.mean(t * np.log(Y) + (1 - t) * np.log(1 - Y)) #funao loss = cross-entropy boa para classificaao

    return  Y, loss, (dU, dV, dW, Eu, Ev, Ew)  #retorna o valor da funao custo e o gradiente do erro em funao dos pesos e bias

O cdigo a seguir implementa a rotina de treinamento do modelo no dataset em amostras, ou batches. Aqui, chamamos a funo train em cada amostra e acumulamos os gradientes durante um batch. Atualizamos ento os pesos do nosso modelo com a subtrao dos gradientes mais um valor de momento para acelerar o treinamento. Este algoritmo uma das formas mais simples de Stochastic Gradient Descent e variaes mais poderosas esto disponveis na literatura e implementadas e prontas para uso em modulos de amprendizado de mquina.

import time

# Parametros de treinamento
epochs = 100
batch_size = 128
learning_rate = 0.0005
momentum = 0.9
n_hidden = 128  #numero de neuronios nas hidden layers


# Adquire dados
mnist = input_data.read_data_sets('MNIST_data', one_hot=True)
sample = mnist.train.next_batch(1)

#VARIAVEIS DA DNN
n_in = sample[0].shape[1]      #numero de features de entrada, neste caso o numero de pixels da imagem
n_out = sample[1].shape[1]      #numero de sadas do grapho, neste caso 10 classes diferentes
n_samples = mnist.train.num_examples - 1     #numero de exemplos no dataset

params = mlp(n_in, n_hidden, n_out)    #inicializa variaveis da rede neural

# Treinamento
print("Start Training Session")

train_err = []
train_accu = []
eval_err = []
eval_accu = []

for epoch in range(epochs):
    
    X, Y = mnist.train.next_batch(batch_size)
    X_eval, Y_eval = mnist.validation.next_batch(batch_size)
    
    err = []                                # placeholder da lista de errors
    upd = [0]*len(params)                   # placeholder dos parametros de atualizaao dos pesos
    corr_pred = []                          # Bookkeeping
    t0 = time.clock()                       # clock para contar o tempo de treinamento
    
    for i in range(batch_size):             # realiza treinamento
        pred, loss, grad = train(X[i], Y[i], *params) # chama a funao train

        for j in range(len(params)):    # faz o update dos parametros da rede
            params[j] -= upd[j]

        for j in range(len(params)):    # calcula o valor a ser atualizado nos parametros para o proximo ciclo
            upd[j] = learning_rate * grad[j] + momentum * upd[j]    # mi*gradiente+momento*update, gradiente com momento

        # Bookkeeping
        err.append(loss)                                            # adquire o custo em lista de errors
        corr_pred.append(np.argmax(Y[i]) == np.argmax(pred)) # adiciona corretude da prediao em lista
    
    train_err.append(err)
    train_accu.append(corr_pred)
    
     
    if epoch % 20 == 0:
        accuracy = np.mean(corr_pred) * 100     # calcula a precisao
        print("Train Epoch: {0}, Loss: {1}, Time: {2}, Accuracy: {3}%".format(1+epoch, np.mean(err), time.clock()-t0, accuracy))
    
    err = []
    corr_pred = []
    
    for i in range(batch_size):
        Z, pred = predict(X_eval[i], *params)   # chama a funao predict para fazer predioes com entradas
        t_ = np.transpose(Y_eval[i])            # transpoe a matriz de labels real para calcular perdas
        eval_loss = -np.mean(t_ * np.log(Z)     # funao loss = cross-entropy para classificaao
                        + (1 - t_) * np.log(1 - Z))

        # Bookkeeping
        err.append(eval_loss)
        corr_pred.append(np.argmax(Y_eval[i]) == np.argmax(pred))
    eval_err.append(err)
    eval_accu.append(corr_pred)    
    
    if epoch % 20 == 0:
        print("Eval Epoch: {0}, Loss: {1}, Time: {2}, Accuracy: {3} %".format(1+epoch, np.mean(err), time.clock()-t0, np.mean(corr_pred) * 100))
