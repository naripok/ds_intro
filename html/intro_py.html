Instalação das Dependencias¶

Inicialmente, necessrio realizar a instalao das dependencias.

Neste projeto sero utilizados como linguagem de programao o Python 3.5.2, e como IDE o Jupyter Notebook. Por se tratar de um projeto em cincia de dados, sero utilizados os seguintes modulos de python:

  • Numpy, como biblioteca matemtica.
  • Pandas, como ferramenta para anlise de dados.
  • Sklearn, como ferramenta de alto nvel para modelagem.
  • Keras, como ferramenta de alto nvel especifica para o desenvolvimento de redes neurais artificiais.
  • Matplotlib, como ferramenta de visualizao e graficos.
  • Seaborn, como ferramenta de visualizao e graficos.

Os modulos sero instalados em um ambiente virtual de python separado da instalao do sistema, por sanidade. Todos os passos descritos a seguir assumem que o sistema operacional utilizado o Ubuntu 16.04 LTS. Os passos so anlogos para outros sistemas, e uma ampla gama de timos tutoriais de instalao esta disponivel na internet. Alternativamente, existe um pacote chamado Anaconda que oferece a maioria das ferramentas para cincia de dados em python e de instalao facilitada.

Instalando o Python 3.5¶

Primeiro, intalando o python 3. Abra o terminal e digite os seguintes comandos:

$$ sudo apt-get update $ sudo apt-get -y install python3 python-pip python-dev
$ pip3 install --upgrade pip

Feito isso, instale o virtualenv:

$ pip install virtualenv

Crie o diretorio onde deseja instalar o ambiente virtual:

$ mkdir MYENV

Crie o ambiente virtual no diretorio desejado:

$ sudo virtualenv -p python3 MYENV

Ative o ambiente virtual:

$ source MYENV/bin/activate

Pronto. Digite no terminal o seguinte comando para verificar se a instalao foi um sucesso:

$ python -V

Este comando deve retornar a verso do python instalada no ambiente virtual, e deve ser igual a 3.X

Para fechar o ambiente virtual, digite no terminal:

$ deactivate

Para facilitar o uso do virtualenv, pode-se instalar o virtualenvwrapper com o seguinte comando:

$ sudo pip install virtualenvwrapper

Agora, para entrar ambiente de trabalho usa-se o comando:

$ workon MYENV

A documentao esclarece maiores dvidas.

Instalalando o Jupyter¶

Com o pip, o gerenciador de pacotes do python, ja instalado, facil prosseguir com as instalaes. Para instalar o jupyter, digite no terminal enquanto com o ambiente virtual ativado:

(MYENV) $ pip3 install jupyter

Quando finalizada a instalao, o jupyter pode ser acessado de dentro do ambiente virtual com o comando:

(MYENV) $ jupyter notebook

Isto abrir uma janela de brower com o jupyter explorer aberto. No canto superior direito clique em new -> python 3. Este comando abrir um notebook vazio, este seu ambiente de trabalho. A documentao deste projeto tambm muito boa e esclarece maiores duvidas.

Instalando os modulos científicos¶

Com o ambiente virtual ativado, digite no terminal:

(MYENV) $ pip3 install --user matplotlib seaborn numpy pandas sklearn tensorflow keras

Feito isso, voc pode testar suas dependencias abrindo digitando no terminal:

(MYENV) $ python3

Isto abrira uma sesso interativa do python. Digite ento:

>>> import matplotlib
>>> import seaborn
>>> import numpy
>>> import pandas
>>> import tensorflow
>>> import keras
>>> import sklearn
>>> exit()

Caso todos os imports sejam executados sem retornar nenhum erro, sua instalao foi um sucesso.
Ocorrendo algum erro, a documentao de cada um dos componentes contm manual completo de intalao em ingls.

Utilizando o Jupyter Notebook¶

O Jupyter notebook um timo ambiente de trabalho para desenvolvimento de projetos em cincia de dados pelo fato de ser um ambiente python (tambm aceita outras linguagens como julia, R, Ruby, etc...) interativo. O processo de programar dada rotina se torna muito mais fcil quando voc pode conferir parte por parte e saber que cada componente do programa roda sem falhas. Alm disso, o poder de exposio do jupyter, gerando visualizaes imediatas aps rodar a celula, o torna uma tima ferramente exploratria e demonstrativa. As celulas Markdown melhoram ainda mais seu poder expositivo sendo possvel escrever em Latex, gerando uma ferramenta que produz apresentaes prontas para exposio ao mesmo tempo que realiza o trabalho de programao.

Comandos basicos como "rodar" a celula, parar, adicionar mais celulas e resetar o ambiente esto disponveis na barra de ferramentas. Alm disso, o jupyter conta com uma poderosa gama de hotkeys para acelerar o trabalho. As principais hotkeys que voc vai utilizar so o Ctrl + Enter, o Shift + Enter e o Alt + Enter, que respectivamente roda a celula selecionada, roda a celula selecionada e seleciona a logo abaixo (abre uma nova caso seja a ltima), e roda a celula e insere uma nova logo abaixo.

Celulas no executadas tem o simbolo [ ] exibido a sua esquerda. Celulas em execuo so sinalizadas por [*], e celulas j executadas so marcadas por seu nmero de execuo [1].

ax3.set_ylabel("Qualitative")

plt.tight_layout(h_pad=3)
